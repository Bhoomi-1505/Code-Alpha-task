Python Code Alpha – Tasks Repository

This repository contains beginner–friendly Python tasks focused on core programming concepts such as loops, conditionals, functions, dictionaries, and file handling.

 Task 1: Hangman Game

A simple text-based Hangman game where:

- Player guesses letters
- Maximum 6 incorrect attempts
- Uses: "random", loops, strings, lists

File: "hangman.py"

Task 2: Stock Portfolio Tracker

A manual stock tracker that:

- Accepts user input for stock name & quantity
- Calculates investment using a predefined stock-price dictionary
- Optionally saves output to ".txt" or ".csv"

File: "stock_portfolio.py"


 Task 4: Basic Rule-Based Chatbot

A simple chatbot with predefined replies such as:

- “hello” → “Hi!”
- “how are you” → “I’m fine, thanks!”
- “bye” → Exits the program

Uses: "if-elif", functions, loops, input/output.

File: "chatbot.py"


 Technologies Used

- Python 3
- Dictionaries
- Loops & conditionals
- Input/output
